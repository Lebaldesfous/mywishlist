# Projet mywishlist

## 0. Présentation du projet

Le projet **mywishlist** est un projet scolaire réalisé durant le mobule de Programmation Web côté serveur. Dans le cadre de notre formation de deuxième année de DUT Informatique, nous sommes amenés, par groupe de 4, à coder, en PHP, le projet mywishlist.

Ce projet doit être écrit dans le langage [PHP](https://www.php.net/docs.php), un langage de programmation web côté serveur très vastement utilisé aujourd'hui. Grâce à ce langage, nous allons ensemble dans ce projet, implémenter des fonctionnalités proposés sur le **sujet**
